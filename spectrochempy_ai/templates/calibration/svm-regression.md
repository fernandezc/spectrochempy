## Scientific Objective

Build a non-linear calibration model using Support Vector Regression
(SVR). SVR maps spectral predictors into a high-dimensional feature
space via a kernel function and finds a regression hyperplane with
controlled tolerance. It is recommended when the relationship between
spectra and the response is non-linear and PLS underperforms.
