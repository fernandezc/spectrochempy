Define the model to fit

.. GENERATED FROM PYTHON SOURCE LINES 61-69

.. code-block:: Python

    def T1_model(t, I0, T1):  # no underscore in parameters names.
        # T1 relaxation model
        import numpy as np

        I = I0 * (1 - np.exp(-t / T1))
        return I
